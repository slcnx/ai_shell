# Changelog

## v0.1.9

- 增强会话管理与同步预览体验，支持双向懒加载、当前 SID 索引同步与弹窗预览修复
- 新增 AI Team / Company Mode 相关页面与后端能力，完善多角色协作工作流
- 优化终端工作区布局、会话虚拟列表与解析/扫描配置能力

## v0.1.8

- 优化主页会话预览布局，修复状态进度条超出预览块的问题
- 继续优化主页会话列与同步中心预览的最新窗口、懒加载与状态联动
- 修复同步中心中会话预览、完整消息入口和选择状态显示的一致性问题

## v0.1.7

- 修复 release workflow 中 macOS 安装依赖阶段的 npm registry 兼容问题
- 在 CI 中统一将 lockfile 里的镜像源规范化为 npm 官方源，提升 esbuild 安装稳定性

## v0.1.6

- 优化同步中心会话/消息选择语义，区分整会话选择与手工消息选择
- 增强同步预览：支持最新 10 条起步、向上/向下懒加载、回到起始/最新
- 修复同步预览点击卡顿与错误勾选显示问题
- 优化复制/同步逻辑，按当前真实待同步消息执行
- 新建终端支持高级配置折叠与原始消息样本路径选择
- 新建同 provider 终端时优化缓存显示与后台预热策略
- 后端预览接口支持分页参数并加入按会话缓存

## v0.1.5

- 发布同步中心与预览体验优化版本
